Info about Gaming Bucket

As said on the Home Page, Gaming Bucket is four highschoolers who are learning to code as a team by using applications like GitHub and Atom to create and share code to create websites with a JavaScript magic 8ball, or browser based video games from JavaScript. But what are the members goals of the company? And how will they achive their goals?

Goals

The Goal of Gaming Bucket was for all the members to learn coding languages like c# to create video games in applications like Unity so that the members can all get jobs as game developers or keep Gaming Bucket as their own company and design games together. But with one member being a senior, going to college starting in the fall of 2020, and the other members being juniors, its very unlikely that the four members will stay together. But in the one year they were working together, the four members did create some projects, for an example, they create an JavaScript browser fighting game were one can create their own character. The project was cancelled due to the COVID-19 out break.

Goals meet

Again, due to the outbreak, and due to no one knowing a real programming language before, Gaming Bucket's members were not able to learn to create games in unity in class. But Mr. Smith created a Game dev club that two of Gaming Bucket's members. The two members that went are Gregory and Andrew. The game they were making was never finished. The game they were making was supposed to be a vr game with swinging mechanics. But the four members have learn JavaScript and they have learn how to work together as a team.
